# ESP8266 WiFi舵机控制工程（v1.0）

## 项目概述
本项目使用ESP8266开发板和SG90舵机，实现通过WiFi网络进行远程控制。支持电脑网页控制。

## 版本信息
- **固件版本**: v1.0
- **发布日期**: 2026-03-22
- **功能特性**: WiFi网页控制

## 硬件需求
- ESP8266开发板（如NodeMCU）
- SG90舵机
- 杜邦线若干
- USB数据线（用于烧录和供电）

## 软件需求
- Arduino IDE
- ESP8266开发板支持包
- 所需库：
  - ESP8266WiFi
  - ESP8266WebServer
  - Servo

## 使用方法

### 网页控制
1. 将ESP8266连接到电脑，烧录固件
2. 打开串口监视器（波特率115200）
3. 等待ESP8266连接WiFi，串口会显示IP地址
4. 在浏览器中输入显示的IP地址
5. 使用网页界面控制舵机：
   - 滑块控制（0-180度）
   - 预设按钮（0°、45°、90°、135°、180°）
   - 手动输入角度

## 串口输出信息
ESP8266启动后会输出以下信息：
- WiFi连接状态
- 设备IP地址
- 舵机控制日志

## 注意事项
- 舵机信号引脚连接到D4（GPIO2）
- 确保舵机有足够的电源供应
- WiFi名称和密码区分大小写

## 故障排除
- **WiFi连接失败**: 检查WiFi名称和密码是否正确
- **舵机不转动**: 检查接线是否正确，电源是否充足
- **网页无法访问**: 确保电脑和ESP8266在同一网络，检查防火墙设置
